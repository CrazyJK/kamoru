border-color 는 테두리의 폭을 지정하는 속성입니다.
좀도 상세하게는 border-top-color, border-right-color, border-bottom-color, border-left-color 각각의 색을 설정할 수 있습니다.
변수값을 지정하는 방법은 하나 지정하는 방법과 shorthand 방법을 이용하여 네개를 한꺼번에 지정하는 방법이 있습니다
