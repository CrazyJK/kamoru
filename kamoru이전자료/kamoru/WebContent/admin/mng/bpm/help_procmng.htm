프로세스의 상태 변경, 삭제를 할 수 있는 관리 화면

 

1. Method 

Date only : 날자로 검색

PROCID : procid 로 검색

ORGPROCDEFID : oegprocdefid 로 검색

2. PROCESS STATE

  검색할 프로세스 상태 선택

ALL : 모두

Running : 실행중

Error : 오류

Overdue : 지연

Suspended : 보류

Terminated : 미완종료

Completed : 완료

Dead : 멈춤

3. CONDITION

  method 에서 선택된 방법에 따른 조건 입력

날자 : Date only 선택시

ID : procid 또는 orgprocdefid 선택시

4. Action

VIEW : 상태 변경 없이 대상 프로세스만 확인

DELETE : 프로세스 삭제

suspend : 보류

resume : 재게

close : 종료

5. Custom Query

    복잡한 조건등으로 위 조건만으로 찾을 수 없을 경우 직접 쿼리를 입력

Use : 사용할 경우 체크(method,process state, condition의 내용은 무시된다.)

6. RESULT

    Action에 대한 결과 화면
